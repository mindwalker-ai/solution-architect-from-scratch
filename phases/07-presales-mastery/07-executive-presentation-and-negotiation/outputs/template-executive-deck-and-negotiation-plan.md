# Executive Deck + Negotiation Plan — Template

> Fill this in for any board-level close. Every figure on the investment slide must be **cited from an earlier artifact** (sizing, BOM, ROI model) — never recalculated here. If you can't name the source lesson or document for a number, don't put it on the slide yet.

## Part 1 — The 12-Slide Deck Outline

| # | Slide | Fill in |
|---|---|---|
| 1 | **The Ask** | One sentence: "Approve **[amount]** (banded **[low–high]**) to **[what]**, over **[timeframe]**, in **[N] waves/phases**, targeting **[the headline outcome metric]**." |
| 2 | Why now | The compelling event / cost of the status quo, in the customer's own words from discovery. |
| 3 | The cost of "do nothing" | Quantify the status-quo trajectory vs the target outcome; name the competitive alternative (rival vendor or no-decision). |
| 4 | Target architecture (ONE slide) | The "money diagram" only — no LLD detail. Cite the HLD it's drawn from. |
| 5 | Investment & ROI (ONE slide) | Headline number + band, payback period, target outcome metric — every figure cited to its source artifact. |
| 6-7 | Risk & sequencing | The wave/phase plan; name the single highest-risk cutover and its gating condition. |
| 8-9 | Competitive edge | 2-3 differentiators vs the named alternative, pulled from the objection/battlecard work. |
| 10 | Objections pre-empted | The two hardest questions this room will ask, answered before they're asked. |
| 11 | The ask, restated | Identical wording to slide 1. |
| 12 | Next step & decision date | The specific date/action being requested. |

### Time budget for the slot

| Slides | Minutes | Cumulative | Note |
|---|---|---|---|
| 1 | | | Land the ask once, don't rush it |
| 2-3 | | | They know their own pain — remind, don't rediscover |
| 4 | | | One clean pass, no detail rabbit holes |
| 5 | | | Slow down — expect the first hard question here |
| 6-7 | | | The gating condition earns its own minute |
| 8-9 | | | Don't rush the "why us" |
| 10 | | | Pre-empt, don't relitigate |
| 11-12 | | | Restate the ask verbatim |
| Q&A buffer | | | Protected — never borrowed from to save an earlier slide |

## Part 3 — Likely-Asks Worksheet

For each ask you expect (from qualification and objection-handling prep), fill in the interest behind the position, the lever you'll trade, and what you ask for in return.

| Likely ask (their position) | The real interest behind it | What you trade (the lever) | What you ask for in return |
|---|---|---|---|
| _e.g., "cut scope for a lower price"_ | | | |
| _e.g., "compress the timeline"_ | | | |
| _e.g., "match a competitor's price"_ | | | |

**Whose lane is each ask in?** Mark **SA** (scope/sequencing/technical) or **AE** (price/terms/contract) for each row above. Never answer an AE-lane ask as if it were yours to trade.

## Part 4 — Reading the Room

| Role | Who (name/title) | Likely behavior | Your move |
|---|---|---|---|
| Economic buyer | | Asks about the number and the downside; decides | Answer with the ROI slide, protect their time |
| Champion | | Pre-briefed; may pre-empt objections for you | Let them field technical follow-ups |
| Skeptic / detractor | | Asks the hardest question, often early | Answer directly and briefly, move on |
| Influencer | | Quiet; the economic buyer checks their face | Watch them during the risk slide |

---

## Part 5 — Pre-Flight Checklist

- [ ] Can I state the ask, from memory, in one sentence identical to slides 1 and 11?
- [ ] Do I know who the economic buyer is, and have I budgeted time toward them?
- [ ] Can I cite the source artifact and assumption behind every number on the investment slide?
- [ ] Is the flexible/protected table filled in and shared with the AE before the meeting?
- [ ] Do I know exactly what I hand to the AE the instant a price or contract-term ask surfaces?
